# The onboarding wizard

The first time someone signs in to a brand-new OpenTMS tenant, the **onboarding wizard** appears. It walks you through a handful of one-time setup steps so the system is ready for daily use.

You only see the wizard **once per tenant** — typically the very first user (the administrator) goes through it. Once the wizard is finished, regular users won't see it again.

:::info Filling this page in later
The detailed step-by-step walkthrough of every wizard screen will be added in a later phase. For now this page describes what the wizard is for and what to expect.
:::

## What the wizard sets up

The exact screens depend on which modules your organization has licensed, but typically the wizard collects:

- **Company details** — name, address, VAT number, default currency, time zone.
- **First branch / organization unit** — most companies start with one and add more later.
- **First user(s)** — at least one administrator account on top of the one you signed in with.
- **Default vehicle types and units** — kilograms vs. tonnes, kilometres vs. miles, etc.
- **Number ranges** — how transport files, orders and invoices should be numbered.
- **Connections to integrations** — optional: link your accounting package, fleet management system, or telematics provider.

## Before you start

The wizard takes about **10–15 minutes** if you have everything to hand. Have these ready:

- Your company's legal name, address and VAT/Chamber of Commerce number
- The list of administrator users you want to create
- Login credentials for any external systems you want to connect

## Skipping the wizard

You can **skip** any optional step and come back to it later from the **Settings** module. The mandatory steps (company details, first user) cannot be skipped — the wizard won't let you finish until they are filled in.

## After the wizard

When you click **Finish** on the last screen, the wizard closes and OpenTMS reloads. You're then taken to the **Dashboard**, ready to start using the system.

If you ever need to revisit the settings configured in the wizard, you can find them in **Settings → General**.
