# Navigating OpenTMS

Every screen in OpenTMS shares the same basic layout. Once you recognize the parts, you can find your way around any module — even ones you've never opened before.

<figure>

![App shell with header, side menu and content area — placeholder](./img/01-app-shell.png)

<figcaption>The OpenTMS interface. ① App bar  ② Module menu  ③ Content area</figcaption>
</figure>

| Area | What it does |
|---|---|
| ① **App bar** (top) | Logo, app menu, search, notifications, chat, language switcher, user menu. The app bar is the same on every screen. |
| ② **Module menu** (left) | Navigation items for the **module you are currently in**. Changes when you switch modules. |
| ③ **Content area** (middle) | The actual screen you opened — a list, a form, a dashboard or a workspace. |

## Read this section in order

1. **[The app bar](./app-bar)** — everything along the top of the screen, including how to switch between modules.
2. **[Your user menu](./user-menu)** — your personal account: profile, password, language, sign-out.
3. **[Notifications](./notifications)** — how OpenTMS keeps you informed of things you care about.
4. **[Chat](./chat)** — the in-app chat for talking to colleagues without leaving OpenTMS.

:::tip
Most of what you see in OpenTMS depends on **your permissions**. If a button or menu item described here is not visible to you, your account most likely does not have access. Contact your administrator to request it.
:::
