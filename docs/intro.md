## How to read this guide

Each chapter is built the same way:

1. **An overview** — what the screen is for, in one paragraph.
2. **Step-by-step tasks** — "How do I…?" walkthroughs you can follow along with.
3. **A reference** — every button, filter and column explained.

Along the way you'll see:

- **Bold** for buttons, menu items and labels you click.
- _Italics_ for the names of screens and sections.
- <kbd>Ctrl</kbd> + <kbd>S</kbd> for keyboard shortcuts.
- :::tip blocks for shortcuts and pro tips.
- :::warning blocks for things to watch out for.
