# App navigation

OpenTMS is a multi-module application: planning, transport files, finance, track & trace, and more all live inside the same shell. This chapter explains how to **sign in**, **find your way around** and **switch between modules**.

If you are new to OpenTMS, read these pages in order:

1. [Signing in](./signing-in)
2. [The app shell](./app-shell)
3. [The main menu](./main-menu)
4. [Your user menu](./user-menu)

:::tip
Most of what you see in OpenTMS depends on **your permissions**. If a button or menu item described in this guide is not visible to you, your account most likely does not have access. Contact your administrator to request it.
:::
