# Navigeren door OpenTMS

Elk scherm in OpenTMS heeft dezelfde basisindeling. Zodra je de onderdelen herkent, vind je je weg in elke module — ook die je nog nooit eerder hebt geopend.

<figure>

![App-omgeving met balk, modulemenu en inhoudsgebied — voorbeeld](./img/01-app-shell.png)

<figcaption>De OpenTMS-interface. ① App-balk  ② Modulemenu  ③ Inhoudsgebied</figcaption>
</figure>

| Onderdeel | Wat het doet |
|---|---|
| ① **App-balk** (boven) | Logo, app-menu, zoekbalk, notificaties, chat, taalkiezer, gebruikersmenu. De app-balk is op elk scherm hetzelfde. |
| ② **Modulemenu** (links) | Navigatie-items voor de **module waarin je nu zit**. Verandert wanneer je tussen modules schakelt. |
| ③ **Inhoudsgebied** (midden) | Het scherm dat je geopend hebt — een lijst, een formulier, een dashboard of een werkomgeving. |

## Lees deze sectie in volgorde

1. **[De app-balk](./app-bar)** — alles wat bovenaan het scherm staat, inclusief schakelen tussen modules.
2. **[Je gebruikersmenu](./user-menu)** — je persoonlijke account: profiel, wachtwoord, taal, uitloggen.
3. **[Notificaties](./notifications)** — hoe OpenTMS je op de hoogte houdt van wat er voor jou belangrijk is.
4. **[Chat](./chat)** — de in-app chat om met collega's te overleggen zonder OpenTMS te verlaten.

:::tip
Wat je in OpenTMS ziet, hangt grotendeels af van **jouw rechten**. Als een knop of menu-item dat hier wordt beschreven niet zichtbaar is, heeft je account waarschijnlijk geen toegang tot die functie. Vraag je beheerder om hem aan te zetten.
:::
